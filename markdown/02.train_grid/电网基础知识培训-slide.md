# 1. 为什么会有电网

---

## 电力的产生与消费特性

```mermaid

    A[发电厂<br/>偏远地区] --> B[传输损耗大<br/>需升压]

    style A fill:#d5e8d4
    style C fill:#f8cecc
    style D fill:#dae8fc
```

---

## 核心原因

- **发电集中化**
  - 大电厂效率高、成本低
  - 但位置偏远（水电站、煤矿、沿海风电）

- **用电分散化**
  - 用户集中在城市、工业区
  - 远离发电厂

- **供需实时平衡**
  - 电能无法大规模储存
  - 发用需实时平衡

- **资源共享**
  - 互济互备，提高可靠性

# 3. 三相交流电原理

---

## 什么是三相交流电？

```mermaid
graph TD
    subgraph 发电机
        A1[线圈A<br/>0°] -->|相差120°| B1[线圈B<br/>120°]
        B1 -->|相差120°| C1[线圈C<br/>240°]
        C1 -->|相差120°| A1
    end

    A1 -->|输出| A2[A相: sin(ωt)]
    B1 -->|输出| B2[B相: sin(ωt-120°)]
    C1 -->|输出| C2[C相: sin(ωt+120°)]
```

**定义**：三个频率相同、幅值相等、相位互差120°的交流电

---

## 为什么三相平衡时总功率恒定？

```
功率瞬时值公式：
P_A = V·I·sin(ωt)·sin(ωt)
P_B = V·I·sin(ωt-120°)·sin(ωt-120°)
P_C = V·I·sin(ωt+120°)·sin(ωt+120°)

总功率 P_total = P_A + P_B + P_C

经过三角函数化简：
P_total = 1.5 × V × I = 常数！
```

---

## 实际意义

- ✅ 电机运转平稳，无脉动
- ✅ 减少机械振动
- ✅ 延长设备寿命

---

## 三相电优势

```mermaid
mindmap
  root((三相交流电))
    优势
      材料节省
        相同功率下<br/>导线用量减少25%
      功率恒定
        电机运转平稳<br/>无转矩脉动
      可产生旋转磁场
        电机结构简单<br/>无需启动装置
      两种电压
        可接星形/三角形<br/>适应不同需求
```

# 4. 相电压与线电压

---

## 连接方式对比

```mermaid
graph LR
    subgraph 星形连接 Y
        A[端子A]
        B[端子B]
        C[端子C]
        N[中性点]

        A --- N
        B --- N
        C --- N

        A -.线电压.-> B
        B -.线电压.-> C
        C -.线电压.-> A
    end

    subgraph 三角形连接 Δ
        A'[端子A]
        B'[端子B]
        C'[端子C]

        A' --- B'
        B' --- C'
        C' --- A'
    end
```

---

## 电压关系

| 参数 | 星形连接(Y) | 三角形连接(Δ) |
|------|-------------|---------------|
| 相电压 U_phase | U_phase = U_line / √3 | U_phase = U_line |
| 线电压 U_line | U_line = √3 × U_phase | U_line = U_phase |
| 线电流 I_line | I_line = I_phase | I_line = √3 × I_phase |

**记忆口诀**：
- 星形(Y)：线电压是相电压的√3倍
- 三角形(Δ)：线电压等于相电压

---

## 实际例子

```
380V/220V低压系统：
- 线电压：380V（两相之间）
- 相电压：220V（火线与零线之间）

验证：380V / √3 ≈ 220V ✓
```

# 5. 有功、无功、功率因数

---

## 推车工作模型

```
┌─────────────────────────────────────────────┐
│                                             │
│  推重物场景：工人推着一辆载重货车工作         │
│                                             │
│  ┌─────────────────────────────────────┐    │  有功功率 (P)
│  │  ────────▶  推车前进（真正做功）     │    │  = 推车前进的能量
│  │         重物位移 100米              │    │  产生实际效益
│  └─────────────────────────────────────┘    │
│                                             │
│  ┌─────────────────────────────────────┐    │  无功功率 (Q)
│  │  ↔ 维持身体平衡（调整重心）         │    │  = 维持状态所需的能量
│  │  来回摆臂但位置不变                  │    │  电感/电容储能-放能循环
│  │  不推车前进，但必须做这些动作        │    │  建立磁场/电场，维持电压
│  └─────────────────────────────────────┘    │
│                                             │
│  工人总体能力 S = √(P² + Q²)                │  视在功率 (S)
│  = 能提供的最大力量                         │  = 设备的额定容量
│                                             │
│  功率因数 cosφ = P/S                        │  = 推车占比
│  = 推车工作占总能力的比例                   │  = 电能利用效率
│                                             │
└─────────────────────────────────────────────┘
```

---

## 定义对比

| 类型 | 符号 | 单位 | 含义 | 类比 |
|------|------|------|------|------|
| 有功功率 | P | 千瓦(kW) | 做有用功的能量 | 推车前进 |
| 无功功率 | Q | 千乏(kvar) | 电感/电容储能-放能 | 维持平衡 |
| 视在功率 | S | 千伏安(kVA) | 总容量能力 | 工人总能力 |
| 功率因数 | cosφ | 无 | 利用效率 | 推车占比 |

---

## 功率三角形

```
        │
        │ Q（无功）
    S   │
    ────┼────
       φ│
        └──── P（有功）

S² = P² + Q²
cosφ = P/S
```

---

## 功率因数的意义

```
cosφ = P / S = 有功 / 视在功率

意义：衡量电能利用效率
- cosφ = 1：100%利用，理想状态
- cosφ = 0.9：90%利用，常见水平
- cosφ < 0.9：效率低，可能罚款
```

---

## 为什么要关注功率因数？

```mermaid
graph LR
    A[低功率因数] --> B[线路电流增大]
    B --> C[线路损耗增加<br/>I²R变大]
    C --> D[设备容量浪费]
    D --> E[电费增加<br/>力调电费]
```

**提升方法**：安装电容器组补偿无功

```mermaid
    UHV1 --> EHV3
    EHV1 --> HV2
    HV1 --> MV2

```

---

## 典型等级与用途

| 电压等级 | 用途 |
|----------|------|
| 1000kV | 跨大区远距离输电（如西电东送） |
| 500kV | 省级主网架、大型电站送出 |
| 220kV | 地区主干网 |
| 110kV | 县级供电、大型工厂 |
| 10kV | 配电网主干线 |
| 380V/220V | 居民和商业用户 |

---

## 变压器原理

```
基本结构：
┌──────────────────────────────────────┐
│                                      │
│   一次绕组 N1     铁芯     二次绕组 N2
│   ┌──────┐      ╱╲╱╲╱╲      ┌──────┐
│   │  │   │     ╱        ╲     │  │   │
│   │  │   │    ╱   铁      ╲    │  │   │
│   │  │   │   ╱    芯       ╲   │  │   │
│   └──────┘  ╱              ╲  └──────┘
│   输入U1    ╲              ╱   输出U2
│             ╲_____________╱
│                                      │
└──────────────────────────────────────┘

变换关系：
U1/U2 = N1/N2  （电压与匝数成正比）
I1/I2 = N2/N1  （电流与匝数成反比）
P1 = P2        （功率守恒，理想情况）
```

---

## 变压器作用

- **升压**：降低输电损耗
- **降压**：满足不同用电需求
- **电气隔离**：安全隔离

---

## 典型输电链路

```
发电厂 ──10kV──► 升压变 ──500kV──► 500kV线路 ──500kV──►
降压变 ──220kV──► 220kV线路 ──220kV──► 降压变 ──110kV──►
110kV线路 ──110kV──► 降压变 ──10kV──► 10kV配电网 ──10kV──►
配电变 ──380V/220V──► 用户
```

# 8. 总结

---

## 核心概念回顾

```
电网基本要素：
├── 发电：集中化、多样化
├── 输电：高压/特高压、低损耗
├── 配电：中低压、广覆盖
└── 用电：实时平衡、质量要求

关键电气概念：
├── 三相电：效率高、功率恒定
├── 电压等级：适应不同传输距离
├── 功率：有功做功、无功建场、功率因数衡量效率
└── 变压器：电磁感应、电压变换
```

## 常用速查

### 功率公式
```
有功功率：P = √3 × U × I × cosφ
无功功率：Q = √3 × U × I × sinφ
视在功率：S = √3 × U × I
功率因数：cosφ = P/S
```

### 电压关系
```
星形连接：U_line = √3 × U_phase
三角形连接：U_line = U_phase
```

### 变压器关系
```
U1/U2 = N1/N2
I1/I2 = N2/N1
```
